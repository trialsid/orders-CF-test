Project: Order.Ieeja grocery storefront (Cloudflare Pages with Pages Functions).
Stack: React + Vite frontend rewritten with Tailwind design system (green accents, light/dark theme toggle), lucide-react icons, TypeScript hooks/components, Cloudflare Pages Functions backend.
Structure:
- src/: React app with modular components (Header, ProductsSection, CartSection, ContactSection, etc.), typed hooks (useProducts, useCart), Tailwind index.css.
- public/favicon.svg: custom gradient basket icon.
- functions/: Cloudflare Pages Functions (_worker.js router, products.js, order.js, _inventory.js).
- wrangler.toml: name="order-ieeja", compatibility_date="2025-07-18", pages_build_output_dir="dist".
- package.json scripts: cf:deploy (preview), cf:deploy:prod (production via --branch production), cf:dev (wrangler pages dev dist), typecheck.
Deployment workflow:
- Local dev: npm run dev (Vite) + npm run cf:dev (build + wrangler pages dev dist) using proxy in vite.config.js for /products & /order.
- Preview deploy: npm run cf:deploy or npm run cf:deploy:preview.
- Production deploy: npm run cf:deploy:prod (targets Pages environment "production").
